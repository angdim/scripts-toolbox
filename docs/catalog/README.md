# Scripts Catalog

Тази директория съдържа автоматично генериран каталог на `scripts-toolbox`.

Файлове:

- `scripts-catalog.md` - Markdown енциклопедичен каталог за четене в GitHub/PyCharm.
- `scripts-catalog.json` - машинно четим каталог за автоматизация.
- `removed-scripts.yaml` - ръчно поддържан архив на изтрити/заменени скриптове.

Генериране:

```bash
scripts_catalog generate
```

Проверка дали каталогът е актуален:

```bash
scripts_catalog validate
```
